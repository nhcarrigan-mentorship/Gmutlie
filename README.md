# Gmutlie 💜

> A safe, anonymous space for teens and youth to talk.

Gmutlie is a private, judgment-free community where teens and young people can pour out their hearts, find understanding, and receive real individual support — all completely anonymously. Wellness coaching is provided by the founder, who has a gift for understanding people.

---

## Features

- **Anonymous Story Sharing** — Share what's weighing on you. No name, no email, no identity required.
- **Category Support** — Stories are organized by: Identity & Coming Out, Faith & Spirituality, Career & Future, Fears & Anxiety, Family & Relationships, Mental Health, and Just Need to Talk.
- **Reactions** — Community members can send ❤️ Love, 🤗 Hugs, and 💪 Stay Strong reactions to stories.
- **Search & Filter** — Browse all stories or filter by category.
- **Persistent Storage** — Stories are saved in your browser via `localStorage`.

---

## Tech Stack

- **React 19** with functional components and hooks
- **React Router v7** for client-side navigation
- **React Context + useReducer** for state management
- **Vite** for fast development and bundling
- **CSS Modules** for scoped component styling

---

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```
